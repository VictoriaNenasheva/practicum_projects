## Project Description
This project presents an end-to-end BI solution for analyzing an e-commerce platform, built using SQL, PostgreSQL, and Apache Superset.

The dashboard provides a comprehensive and interactive view of key business metrics, including revenue, user behavior, conversion rates, and sales performance across multiple dimensions.

The solution is designed to simulate a real-world analytics environment, where business stakeholders can monitor performance, identify growth drivers, and make data-driven decisions.

---

## Dashboard Structure

The dashboard consists of three main analytical sections:

### 1. Main Metrics
Provides a high-level overview of business performance:
- Total revenue
- Average order value (AOV)
- Conversion rate
- Revenue per session
- Number of products
- Number of transactions
- Unique users

---

### 2. Sales Analysis
Focuses on revenue dynamics and sales distribution:
- Revenue trends over time
- Sales by acquisition channels (organic, ads, email, referral)
- Revenue breakdown by operating systems
- Sales by product categories
- Top-performing products
- Month-over-month (MoM) growth analysis

---

### 3. User Analytics
Analyzes user behavior and engagement:
- Monthly active users (MAU)
- Active users
- Session duration
- Total session time
- Conversion funnel
- Time to first purchase

---

## Key Features
