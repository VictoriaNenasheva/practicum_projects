# E-commerce Analytics Dashboard

## Project Description
This project presents an end-to-end analytics solution for an e-commerce platform using SQL, PostgreSQL, and Apache Superset.

The dashboard provides a comprehensive overview of business performance, including revenue trends, user behavior, conversion rates, and sales distribution across different dimensions.

The goal of the project is to support data-driven decision-making and identify key growth drivers.

---

## Dashboard Structure

The dashboard is divided into three main sections:

### 1. Main Metrics
- Total revenue
- Average order value (AOV)
- Conversion rate
- Revenue per session
- Number of products
- Unique users

### 2. Sales Analysis
- Revenue trends over time
- Sales distribution by channels (organic, ads, email, referral)
- Revenue by operating systems
- Sales by product categories
- Top-performing products
- Month-over-month (MoM) growth

### 3. User Analytics
- Unique users (MAU)
- Active users
- Session duration
- Total session time
- Conversion funnel
- Time to first purchase

---

## Key Features
- Interactive filters (date, category, OS, month)
- Multi-tab navigation
- KPI monitoring
- Trend analysis
- Drill-down capabilities

---

## Tools & Technologies
- SQL
- PostgreSQL
- Apache Superset
- DBeaver

---

## Insights
- Revenue demonstrates steady growth with a significant increase in the last quarter
- Organic traffic is the primary source of revenue
- Conversion rate improves over time, indicating funnel optimization
- Certain categories (electronics, furniture) show higher growth rates
- User engagement metrics indicate increasing platform activity

---

## How to Use
1. Open the dashboard in Apache Superset
2. Apply filters (date range, category, OS)
3. Navigate between tabs:
   - Main Metrics for overview
   - Sales Analysis for revenue insights
   - User Analytics for behavior analysis
