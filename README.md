# Job Market Analysis (SQL Project)

## Project Description
This project analyzes job vacancies data from hh.ru to identify key trends in the data analyst job market.

The dataset contains information about vacancies, including salary ranges, regions, companies, employment types, work schedules, and required skills.

The project was completed independently as part of portfolio development.

## Objectives
- Analyze salary ranges (average, minimum, maximum)
- Identify regions with the highest number of vacancies
- Find top companies by number of job postings
- Analyze employment types and work schedules
- Explore distribution of experience levels (Junior, Middle, Senior)
- Identify the most in-demand hard and soft skills

## Tools and Technologies
- SQL
- PostgreSQL
- DBeaver

## Dataset
Table: public.parcing_table

The dataset includes:
- vacancy name
- salary range (salary_from, salary_to)
- employer
- region (area)
- employment type
- work schedule
- experience level
- key skills (hard and soft)

## Key Insights

### Salary
- Average salary (from): ~109,525
- Average salary (to): ~153,846
- Salary range varies significantly: from 50 to ~497,500

### Top Regions
- Moscow has the highest number of vacancies
- Saint Petersburg is in second place
- Other regions have significantly fewer вакансий

### Top Employers
- Sber
- Wildberries
- Ozon
- VTB
- Yandex

### Employment Type
- Full-time employment dominates
- Part-time and internship positions are rare

### Work Format
- Office work is the most common
- Remote work has a noticeable share
- Flexible schedules are less common

### Experience Levels
- Junior (1–3 years) — most in demand
- Middle (3–6 years) — second place
- Entry-level (no experience) — limited number of vacancies
- Senior — very small share

### Key Skills

Hard skills:
- SQL
- Python
- Data analysis
- Documentation

Soft skills:
- Attention to detail
- Analytical thinking
- Communication
- Proactiveness

## Conclusion
The job market for analysts is focused primarily on junior and middle-level specialists.

Employers prioritize SQL and Python skills, along with strong analytical thinking and attention to detail.

Most vacancies offer full-time employment with office or hybrid formats, while remote work is also becoming more common.
